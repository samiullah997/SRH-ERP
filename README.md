# 📖 SRH-ERP <a name=""></a>

- **SRH-ERP** is a web application that allows users to manage their ERP system. It allows users to create, read, update and delete their admission, library, inventory, and transport. It also allows users to generate reports for their all departments.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- Java runtime environment
- MySQL
- Spring Boot

### Setup

Clone this repository to your desired folder:

### Install

Install this project with:

- Clone the repository using `git@github.com`
- Navigate to the folder using `cd SRH-ERP`
- Use the following commands to
  
  ```sh
    mvn clean install
  ```
    
    ```sh
      mvn spring-boot:run
    ```


### Usage

To run the project, execute the following command:

```sh
mvn spring-boot:run
```

### Run tests

To run tests, run the following command:

```sh
mvn test
```
